# Hello World Python Project

## Overview
A minimal Python "Hello World" application.

## Stack
- **Language:** Python 3.12
- **Environment:** Replit (Nix, stable-25_05 channel)

## Project Structure
- `hello-world.py` - Main entry point, prints "Hello World!"

## Running the App
The workflow "Start application" runs `python hello-world.py` as a console process.
